How Emmy Helps States Determine Medicaid Eligibility

Under HR1, Medicaid expansion adults must meet community engagement requirements in addition to income eligibility. This demo walks through how the state system and Emmy work together to verify both.

The eligibility process

When someone applies for or renews Medicaid, states follow these steps:

State system
Emmy
Outcome
1
Application or renewal submitted
The person submits their application or renewal form, including income and community engagement self-attestation.
State system
2
Income eligibility verified
The state checks income against the Medicaid threshold using federal and state data sources (FDSH, Work Number, SWICA).
State system
3
Exemption screening
The system checks if the person qualifies for a CE exemption (pregnancy, disability, caregiving, tribal status, and others).
State system
4
Ex parte CE verification attempted
The state tries to confirm community engagement activities using electronic data. This often fails due to gaps in available data.
State system
5
Emmy activates
The person receives a link to Emmy, where they report and verify their activities: connect payroll, document education, log volunteer hours.
Emmy
6
Determination
The caseworker reviews income, exemption, and CE data (including Emmy's report) and makes an eligibility decision.
