Emmy API

NSC call via Emmy API

The National Student Clearinghouse query is an Emmy API call — not a native state system check. It augments what the state already has.

  • State has basic enrollment data; Emmy API calls NSC for live credit hours & enrollment intensity
  • NSC returns below-half-time status → triggers CE flag
  • Without Emmy API, this check doesn't exist in the state's system
  • Emmy then activates to collect supplemental documentation
